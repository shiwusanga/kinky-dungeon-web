游戏内容 · What awaits in the dungeon

你被困在一座不断向下延伸的地牢之中。每一层都由程序生成，充满敌人、陷阱、上锁的门与隐藏房间。这不是靠数值碾压的游戏——而是靠观察、走位与挣脱求生。

🗺️

程序化地牢

多层随机地图，房间、走廊、陷阱与宝藏每次都不重样，耐刷不腻。

⚔️

回合制战斗

精细的回合制对抗，讲究走位、时机与道具配合，而非无脑堆数值。

招牌机制：束缚 & 挣脱 核心玩法

本作最鲜明的张力来自束缚系统：敌人不会简单地把你打死，而是用绳索、锁链、蛛网将你束缚在原地。一旦被缚，你的移动与行动将大受限制，处境急转直下。你必须通过挣扎、借助道具或利用环境挣脱束缚，再抓住空隙反击或脱身。它把「被抓住」变成了持续的心理博弈，是整款游戏紧张感的灵魂。

代码特色 · Engineering

一款纯网页、零后端、即开即玩的网页游戏背后的工程实现。

🎨

PixiJS 渲染

基于 WebGL / Canvas2D 的高性能 2D 渲染，流畅绘制大量精灵与动画。

🧩

web-shim.js 桥接

桥接窗口、全屏与 Mod 接口，让网页版获得接近原生的体验（纯网页构建不含桌面 mod 系统）。

📦

纯静态部署

GitHub Pages 静态托管，零后端、零安装，打开链接即玩。

📱

viewport 自适应

启用 viewport 缩放，桌面键鼠与移动端触屏无缝衔接。

⏳

渐进式加载

约 11MB 主脚本首屏懒加载，配加载界面平滑过渡，避免白屏。

🗂️

模块化架构

Game / Scripts / Mods / assets / TextureAtlas 分层组织，结构清晰易扩展。
